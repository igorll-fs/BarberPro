💈 Aplicativo completo para gestão de barbearias. React Native + Firebase. 3 perfis: cliente, dono e funcionário. Agendamentos, chat, avaliações e muito mais.

## Opção 2: Média
```
💈 BarberPro - Sistema completo de gestão para barbearias com React Native, Expo e Firebase. Features: agendamentos em tempo real, chat cliente-barbearia, avaliações, promoções, onboarding e notificações push.
```

## Opção 3: Completa (About section)
```
💈 Aplicativo profissional para gestão de barbearias

• 📱 App mobile (iOS/Android) em React Native + Expo SDK 54
• 🔥 Backend serverless com Firebase (Auth, Firestore, Functions, Storage)
• 👥 3 perfis de usuário: Cliente, Dono e Funcionário
• 📅 Sistema de agendamentos com confirmação em tempo real
• 💬 Chat integrado cliente-barbearia
• ⭐ Sistema de avaliações e reviews
• 🎉 Gestão de promoções e descontos
• 🔔 Notificações push (FCM)
• 📊 Dashboard com métricas para donos
• 🔐 Autenticação segura com WhatsApp OTP (Twilio)

Pronto para importação no Firebase Studio 🚀
```

## 🎯 Como Adicionar no GitHub

1. Vá na página do repositório: `github.com/SEU_USUARIO/barberpro`

3. Preencha:

4. Salve!
